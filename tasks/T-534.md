# DELETE /stocks/:id 実装

ストック削除エンドポイント DELETE /api/stocks/:id を実装してください。

## 前提

- docs/stock-api-spec.md の仕様に従う
- T-502（認証ミドルウェア）が実装済みであること

## やること

1. DELETE /api/stocks/:id を実装する:
   - パスパラメータから stock ID を取得する
   - 認証ユーザーの stock かチェックする（所有権）
   - 関連する memos レコードを削除する
   - stocks レコードを削除する
   - 204 No Content を返す
   - 所有者でない / 存在しない場合は 404

2. `npm test` で全テストがパスすることを確認する

3. 変更をコミットする
