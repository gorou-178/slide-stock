# ストック一覧画面構築

stocks.astro を本格的なストック一覧画面に構築してください。

## 前提

- docs/ui-spec.md の仕様に従う
- T-532（GET /api/stocks）が実装済みであること
- T-551（BaseLayout）が実装済みであること

## やること

1. src/pages/stocks.astro を更新する:
   - BaseLayout を使用する
   - ページ読み込み時に GET /api/stocks を呼び出す
   - ストックカードの一覧を表示する:
     - タイトル
     - サムネイル画像
     - プロバイダバッジ
     - メモ抜粋
     - 元スライドへのリンク
   - ローディング状態の表示
   - 空状態の表示（「まだスライドがストックされていません」等）
   - エラー状態の表示

2. URL 入力フォームをページ上部に配置する（送信ロジックは T-553 で実装）

3. `npm run test:e2e` で全テストがパスすることを確認する

4. 変更をコミットする
