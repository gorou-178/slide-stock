# Google OIDC 認証のテスト作成

Google OIDC 認証フローのユニットテストを作成してください。

## やること

1. JWT 検証ロジックのテストを作成する
   - 正しい Google ID Token を検証できること
   - 期限切れトークンを拒否すること
   - 不正な署名のトークンを拒否すること
   - 必須クレーム (sub, email, name) が含まれていること
2. `GET /me` エンドポイントのテストを作成する
   - Bearer トークン付きリクエストでユーザー情報を返すこと
   - トークンなしで 401 を返すこと
   - 不正トークンで 401 を返すこと
   - 初回アクセス時にユーザーが自動作成されること
   - 既存ユーザーの場合はそのユーザー情報を返すこと
3. セッション管理のテストを作成する
   - セッションが正しく発行されること
   - セッション有効期限が機能すること

## テストデータ

- テスト用 JWT を jose ライブラリ等で生成する
- Google の JWKS エンドポイントはモックする

## 注意

- テストはこの段階では失敗してよい（Dev が後で実装する）
