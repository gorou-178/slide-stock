# 詳細画面の実装

QA が作成したテスト (044-test-ui-detail) を通すよう、詳細画面を実装してください。

## やること

1. 詳細ページ (`src/pages/stocks/[id].astro`) を作成する
   - `GET /stocks/:id` で stock 情報を取得
   - 404 ハンドリング
2. embed 表示を実装する
   - embed_url を使った iframe 埋め込み
   - プロバイダに応じた適切なサイズ設定
   - status=pending の場合は「メタデータ取得中」を表示
   - status=failed の場合は「取得に失敗しました」を表示
3. メモ編集機能を実装する
   - `GET /stocks/:id/memo` で既存メモを取得
   - テキストエリアでメモを編集
   - 保存ボタンで `PUT /stocks/:id/memo` を呼び出し
   - 保存成功時にフィードバック表示
4. ナビゲーションを実装する
   - 一覧画面への「戻る」リンク
   - 元スライドへの外部リンク (target="_blank")

## デザイン方針

- embed を画面上部に大きく表示
- メモ編集エリアはその下に配置
- レスポンシブ対応

## 確認

- QA が作成した詳細画面テストがすべて通ること
