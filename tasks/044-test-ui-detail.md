# 詳細画面のテスト作成

スライド詳細画面の E2E テストを作成してください。

## やること

1. 詳細画面の表示テストを作成する
   - 指定 stock の詳細情報が表示されること
   - embed が表示されること
   - メモが表示されること
   - 元スライドへのリンクが表示されること
   - 一覧画面への戻るリンクが表示されること
2. メモ編集のテストを作成する
   - メモの入力・保存ができること
   - 保存後にメモが更新されていること
   - 空メモの場合にプレースホルダーが表示されること
3. エラーケースのテストを作成する
   - 存在しないストック ID で 404 ページが表示されること
   - 他ユーザーのストック ID で 404 ページが表示されること

## 注意

- 認証バイパスを使用してテストする
- テストはこの段階では失敗してよい（Dev が後で実装する）
