# Memo API仕様策定

メモ API（PUT/GET）の詳細仕様を定義してください。

## 前提

- CLAUDE.md のAPI設計（セクション6）を参照する
- docs/database.md の memos テーブルを参照する（UNIQUE stock_id: 1 stock に 1 memo）

## やること

1. docs/memo-api-spec.md を作成し、以下を定義する:

   ### PUT /api/stocks/:id/memo
   - リクエストボディ: `{ memo_text: string }`
   - 動作: upsert（INSERT OR REPLACE）
   - レスポンス: 200 + memo オブジェクト
   - バリデーション: memo_text の最大文字数、空文字列の扱い
   - 所有権チェック: stock の所有者のみ操作可能

   ### GET /api/stocks/:id/memo
   - レスポンス: 200 + memo オブジェクト / 404（メモ未作成時）
   - 所有権チェック: stock の所有者のみ取得可能

   ### 共通
   - エラーレスポンス形式: stock-api-spec.md と統一する

2. QA/DEV タスク（T-541〜T-543）が実行可能になるよう、仕様を具体的に記述する
