# プロバイダ検出モジュール実装

URL からプロバイダを検出し、canonical URL に正規化するモジュールを実装してください。

## 前提

- docs/provider-spec.md（T-510 で作成）の仕様に従う
- T-512 の QA テストが先に作成されている前提（TDD）

## やること

1. docs/provider-spec.md を読み、各プロバイダの URL パターンと正規化ルールを把握する

2. worker/lib/provider.ts を作成する:
   - `detectProvider(url: string): { provider: string, canonicalUrl: string } | { error: string }` を実装する
   - SpeakerDeck, Docswell, Google Slides の URL パターンを判定する
   - canonical URL に正規化する（クエリパラメータ除去、末尾スラッシュ統一等）
   - 未対応 URL にはエラーを返す

3. テストを実行して全てパスすることを確認する: `npm test`

4. 変更をコミットする
