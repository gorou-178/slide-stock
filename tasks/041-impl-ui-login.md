# ログイン画面の実装

QA が作成したテスト (040-test-ui-login) を通すよう、ログイン画面を実装してください。

## やること

1. ログインページ (`src/pages/login.astro`) を作成する
   - Google Login ボタンを配置
   - Google OAuth 認証フローを開始するリンク/ボタン
   - シンプルなデザイン（サービス名 + ログインボタン）
2. 認証状態の管理を実装する
   - ログイン済みかどうかのチェック
   - 未ログイン時のリダイレクト処理
3. レイアウトを作成する
   - `src/layouts/Layout.astro` — 共通レイアウト
   - ヘッダー（サービス名、ログアウトボタン）
4. ログアウト機能を実装する
   - セッションのクリア
   - ログイン画面へのリダイレクト

## デザイン方針

- 最小限の CSS（フレームワーク不使用、または軽量 CSS のみ）
- レスポンシブ対応
- アクセシビリティを考慮

## 確認

- QA が作成したログイン画面テストがすべて通ること
