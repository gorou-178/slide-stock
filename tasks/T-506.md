# 環境変数・Secrets設定

Google OIDC と セッション管理に必要な環境変数を設定してください。

## 前提

- docs/auth-spec.md の仕様に従う
- wrangler.toml に現在 D1 バインディングのみ設定済み

## やること

1. .dev.vars に開発用の環境変数を追加する:
   - GOOGLE_CLIENT_ID（Google Cloud Console から取得する値のプレースホルダー）
   - SESSION_SECRET（開発用のランダム文字列）

2. worker/index.ts の Env インターフェースを更新する:
   - GOOGLE_CLIENT_ID: string を追加
   - SESSION_SECRET: string を追加

3. wrangler.toml にコメントで本番用 Secrets の設定手順を記載する

4. `npm test` で全テストがパスすることを確認する

5. 変更をコミットする
