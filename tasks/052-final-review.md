# 最終レビューと要件充足確認

全タスク完了後に MVP の要件充足を確認してください。

## チェックリスト

### 機能要件

- [ ] Google Login (OIDC) でログインできること
- [ ] SpeakerDeck のスライドを URL 入力で登録できること
- [ ] Docswell のスライドを URL 入力で登録できること
- [ ] Google Slides のスライドを URL 入力で登録できること
- [ ] ストック一覧で embed 表示できること
- [ ] 各スライドにメモを残し、閲覧・編集できること
- [ ] 元スライド URL へのリンクからアクセスできること
- [ ] ストックを削除できること

### 非機能要件

- [ ] 全テスト (Unit + E2E) が通ること
- [ ] TypeScript の型エラーがないこと
- [ ] ビルドが成功すること
- [ ] 秘匿情報がコードにハードコードされていないこと
- [ ] `.env` が `.gitignore` に含まれていること
- [ ] レスポンシブデザインであること

### ドキュメント

- [ ] docs/deploy.md が作成されていること
- [ ] 環境変数一覧が整理されていること

## やること

1. 上記チェックリストを1つずつ確認する
2. 問題があれば修正タスクを tasks/TASKS.md に追加する
3. 全項目クリアで MVP 完成とする
