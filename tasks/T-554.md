# ストック詳細画面構築

スライドの embed 表示とメモ編集ができる詳細画面を構築してください。

## 前提

- docs/ui-spec.md の仕様に従う
- T-533（GET /api/stocks/:id）が実装済みであること
- T-541（PUT /api/stocks/:id/memo）が実装済みであること
- T-534（DELETE /api/stocks/:id）が実装済みであること
- T-551（BaseLayout）が実装済みであること

## やること

1. src/pages/stocks/[id].astro を作成する（動的ルーティング）:
   - BaseLayout を使用する
   - GET /api/stocks/:id からストック情報を取得する
   - embed 表示（EmbedViewer コンポーネント / iframe）
   - メモ編集テキストエリア + 保存ボタン（PUT /api/stocks/:id/memo）
   - 元スライド URL へのリンク
   - 削除ボタン（確認ダイアログ付き → DELETE /api/stocks/:id → /stocks にリダイレクト）
   - 一覧に戻るリンク

2. `npm run test:e2e` で全テストがパスすることを確認する

3. 変更をコミットする
