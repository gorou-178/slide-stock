# ログアウトエンドポイント実装

POST /api/auth/logout エンドポイントを実装してください。

## 前提

- docs/auth-spec.md の仕様に従う
- T-502 の認証ミドルウェアが実装済みであること

## やること

1. POST /api/auth/logout を実装する:
   - セッション Cookie をクリアする（Max-Age=0 の Set-Cookie）
   - D1 セッションテーブルを使用している場合はレコードも削除する
   - 200 OK を返す

2. `npm test` で全テストがパスすることを確認する

3. 変更をコミットする
