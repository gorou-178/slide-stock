# Cloudflare Queues設定

wrangler.toml に Cloudflare Queues の producer/consumer バインディングを追加してください。

## 前提

- docs/oembed-spec.md（T-520 で作成）の Queue メッセージスキーマに従う
- docs/architecture.md の非同期処理フローを参照する

## やること

1. wrangler.toml に Queues 設定を追加する:
   - producer バインディング（API から Queue にメッセージを送信）
   - consumer 設定（Worker が Queue メッセージを受信）
   - Queue 名: slide-stock-oembed-queue（仮）

2. worker/index.ts の Env インターフェースを更新する:
   - OEMBED_QUEUE バインディングの型定義を追加する

3. `npm test` で全テストがパスすることを確認する

4. 変更をコミットする
