# 一覧画面のテスト作成

ストック一覧画面の E2E テストを作成してください。

## やること

1. 一覧画面の表示テストを作成する
   - URL 入力フォームが表示されること
   - ストック一覧が表示されること
   - 各ストックにタイトル、サムネイル、メモ、元スライドリンクが表示されること
2. スライド登録のテストを作成する
   - URL を入力して送信すると、一覧にスライドが追加されること
   - pending 状態の表示が正しいこと
   - 不正な URL で適切なエラーメッセージが表示されること
   - 重複 URL で適切なエラーメッセージが表示されること
3. スライド削除のテストを作成する
   - 削除ボタンを押すとストックが一覧から消えること
   - 削除確認ダイアログが表示されること
4. embed 表示のテストを作成する
   - ready 状態のストックに embed が表示されること（lazy load）
5. 元スライドリンクのテストを作成する
   - リンクをクリックすると元 URL に遷移すること（target="_blank"）

## 注意

- 認証バイパスを使用してテストする
- テストはこの段階では失敗してよい（Dev が後で実装する）
