# E2E テスト基盤の構築

Playwright を使った E2E テストの土台を作成してください。

## やること

1. Playwright の設定ファイル（playwright.config.ts）を作成する
   - baseURL をローカル dev サーバーに設定
   - テスト用のブラウザ設定（chromium のみで十分）
   - スクリーンショット・トレースの保存設定
2. E2E テスト用のディレクトリ構造を作成する
   - e2e/ ディレクトリをプロジェクトルートに配置
   - e2e/fixtures/ にテスト用フィクスチャを配置
3. ヘルスチェック用の最初の E2E テストを作成する
   - トップページにアクセスできること
   - ページタイトルが正しいこと
4. package.json に `test:e2e` スクリプトを追加する

## 確認

- `npm run test:e2e` でテストが実行できること（この段階ではページ未実装のため失敗してよい）
