# デプロイ先最終確定

本番デプロイ先の Cloudflare リソースを最終確定してください。

## やること

1. 以下のリソース名を確定し、docs/production-plan.md に記録する:
   - Cloudflare Pages プロジェクト名
   - Cloudflare Workers 名
   - D1 データベース名（本番用）
   - カスタムドメイン: slide-stock.gorou.dev

2. wrangler.toml の設定を確認する:
   - 現在の設定値が正しいか
   - `[env.production]` セクションに本番リソース名を記載する

3. Cloudflare ダッシュボードで以下を確認（手動作業が必要な場合は手順を記載）:
   - Pages プロジェクトの作成状況
   - Workers の作成状況
   - D1 データベースの作成状況
   - DNS / カスタムドメインの設定状況

4. 未確定事項があれば明記し、判断が必要な項目をリストアップする
