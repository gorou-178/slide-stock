# 一覧画面の実装

QA が作成したテスト (042-test-ui-list) を通すよう、一覧画面を実装してください。

## やること

1. 一覧ページ (`src/pages/index.astro`) を作成する
   - ログイン済みの場合にストック一覧を表示
   - 未ログインの場合はログイン画面にリダイレクト
2. URL 入力フォームを実装する
   - URL 入力テキストフィールド
   - 送信ボタン
   - `POST /stocks` API を呼び出し
   - 成功時に一覧を再描画
   - エラー時にエラーメッセージを表示
3. ストック一覧コンポーネントを実装する
   - `GET /stocks` API を呼び出して一覧取得
   - 各ストックカードの表示:
     - タイトル（status=pending 時は「読み込み中...」表示）
     - サムネイル画像
     - メモ（先頭数行のプレビュー）
     - 元スライドへのリンク (target="_blank")
     - 削除ボタン
   - embed 表示（lazy load、Intersection Observer 活用）
4. スライド削除機能を実装する
   - 確認ダイアログの表示
   - `DELETE /stocks/:id` API 呼び出し
   - 一覧から該当カードを削除

## デザイン方針

- カード型レイアウト
- レスポンシブ対応
- 空状態（ストックゼロ）時のメッセージ表示

## 確認

- QA が作成した一覧画面テストがすべて通ること
