# ベースレイアウト実装

共通レイアウトコンポーネント BaseLayout.astro を実装してください。

## 前提

- docs/ui-spec.md（T-550 で作成）の仕様に従う
- src/layouts/ ディレクトリに .gitkeep のみ存在

## やること

1. docs/ui-spec.md を読み、レイアウト仕様を把握する

2. src/layouts/BaseLayout.astro を作成する:
   - 共通 HTML head（meta, title, viewport）
   - ナビバー（サービス名、ユーザー名表示、ログアウトボタン）
   - メインコンテンツ slot
   - 基本 CSS（docs/ui-spec.md のスタイリング方針に従う）
   - lang="ja" 属性
   - アクセシビリティ: セマンティック HTML、skip navigation リンク

3. 既存ページ（index.astro, login.astro, stocks.astro）を BaseLayout で wrap する

4. `npm run test:e2e` で全テストがパスすることを確認する

5. 変更をコミットする
