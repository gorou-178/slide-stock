# ログイン画面のテスト作成

ログイン画面の E2E テストを作成してください。

## やること

1. ログイン画面の表示テストを作成する
   - ログイン画面が表示されること
   - Google Login ボタンが表示されること
   - 未ログイン状態でアクセスするとログイン画面にリダイレクトされること
2. ログインフローのテストを作成する（認証バイパス使用）
   - テスト用認証でログインできること
   - ログイン後に一覧画面にリダイレクトされること
3. ログアウトのテストを作成する
   - ログアウトボタンが表示されること（ログイン後）
   - ログアウト後にログイン画面に戻ること

## 注意

- Google OAuth の実際の認証フローはテストしない（認証バイパスを使用）
- テストはこの段階では失敗してよい（Dev が後で実装する）
