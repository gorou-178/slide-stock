# GET /stocks/:id/memo 実装

メモ取得エンドポイント GET /api/stocks/:id/memo を実装してください。

## 前提

- docs/memo-api-spec.md の仕様に従う
- T-502（認証ミドルウェア）が実装済みであること

## やること

1. GET /api/stocks/:id/memo を実装する:
   - パスパラメータから stock ID を取得する
   - stock の所有権を確認する
   - memos テーブルから memo を取得する
   - メモが存在する場合: 200 + memo オブジェクト
   - メモが存在しない場合: 404

2. `npm test` で全テストがパスすることを確認する

3. 変更をコミットする
