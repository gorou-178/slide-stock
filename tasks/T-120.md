# PR(main)でCI実行するGitHub Actions workflow追加

main ブランチへの PR で自動テストを実行する GitHub Actions workflow を作成してください。

## やること

1. `.github/workflows/ci.yml` を作成する

2. トリガー:
   - `pull_request` → `main` ブランチ
   - `push` → `main` ブランチ

3. 実行ステップ:
   - Node.js セットアップ (v20)
   - `npm ci` で依存インストール
   - `npx tsc --noEmit` で型チェック
   - `npm test` でユニットテスト実行 (vitest)
   - `npm run build` でビルド確認

4. 注意事項:
   - E2E テスト (`npm run test:e2e`) はこのタスクでは含めない（別途対応）
   - Cloudflare 固有の環境変数は不要（vitest は @cloudflare/vitest-pool-workers で実行される）
   - キャッシュ: `node_modules` をキャッシュして高速化する
