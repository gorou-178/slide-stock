# oEmbed埋め込みコンポーネント実装

oEmbed の embed_url から iframe を生成するコンポーネントを実装してください。

## 前提

- docs/ui-spec.md の仕様に従う
- stocks テーブルの embed_url フィールドを使用する

## やること

1. src/components/EmbedViewer.astro（または .tsx）を作成する:
   - embed_url から iframe を生成する
   - プロバイダ別のアスペクト比・サイズ対応
   - lazy loading（loading="lazy" 属性）
   - レスポンシブ対応（width: 100%, aspect-ratio）
   - status が 'pending' の場合はローディング表示
   - status が 'failed' の場合はエラー表示
   - アクセシビリティ: title 属性、適切な ARIA ラベル

2. ストック一覧画面（T-552）および詳細画面（T-554）から使用できるようにする

3. `npm run test:e2e` で全テストがパスすることを確認する

4. 変更をコミットする
